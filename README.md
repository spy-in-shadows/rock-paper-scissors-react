# Rock Paper Scissors 🪨📃✂️

A fun, beginner-friendly React application for playing Rock Paper Scissors against the computer! This project demonstrates core React concepts like `useState`, list mapping, inline state logic, and conditional rendering.

**Live Demo:** [https://rps-react-spy-in-shadows.netlify.app](https://rps-react-spy-in-shadows.netlify.app)

## Features 🚀

- **Round Selection:** Choose to play 1, 3, or 5 total rounds before the game starts.
- **Real-time Score Tracking:** Keeps a running tally of your wins versus the computer's wins.
- **Match History:** Generates a mapped list displaying a comprehensive timeline of all past moves and round outcomes.
- **Dark Mode Toggle:** A functioning light/dark mode switch built with basic DOM manipulation and CSS.
- **Dynamic UI:** Responsive UI that reveals a final winner based on score when the selected number of rounds are completed.

## Tech Stack 💻

- **React.js** (Bootstrapped with Vite)
- **Vanilla CSS**
- **JavaScript**
- Hosted on **Netlify**

## Getting Started 🛠️

To interact with or edit the project on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/spy-in-shadows/rock-paper-scissors-react.git
   cd "Rock-Paper-Scissors(React)"
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the Vite development server:**
   ```bash
   npm run dev
   ```

4. **Play!**
   Open your browser to `http://localhost:5173/` (or whichever port Vite provides in your terminal) to check it out locally.
